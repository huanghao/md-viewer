# 配置说明

本项目配置的权威说明以 man page 为准：

- `man/md-viewer.1` - 命令行工具使用说明（mdv, mdv-admin）
- `man/md-viewer.config.5` - 配置文件格式说明

补充设计文档：

- `docs/design/config-strategy.md`

## CLI 工具

### mdv (客户端)
用于向运行中的 MD Viewer 添加文件。

```bash
mdv README.md                    # 添加并切换到文件
mdv --no-focus notes.md          # 添加文件但不切换
mdv -p 3001 doc.md               # 指定服务器端口
```

### mdv-admin (管理工具)
用于管理同步记录和查看统计信息。

```bash
mdv-admin stats                  # 查看统计信息
mdv-admin cleanup                # 清理过期记录（>6个月）
```

详细用法见：`man md-viewer` 或 `mdv --help` / `mdv-admin --help`

## 配置变更时的最低要求

1. 更新 `man/md-viewer.config.5`（默认值、范围、兼容性说明）
2. 更新 `man/md-viewer.1`（如涉及 CLI 变化）
3. 同步更新 `docs/design/config-strategy.md`（如有设计变化）
4. 更新配置加载代码
5. 如涉及废弃项，保留向后兼容并在文档标注
