详细交互设计
================================================================================

【刷新按钮状态】

1. 无更新（隐藏）
   ┌──────────────────────────┐
   │  ⚙  │  ↑✓   • 2分钟前    │
   └──────────────────────────┘

2. 有更新（显示橙色圆点）
   ┌──────────────────────────┐
   │  ⚙  │  ↻●  ↑✓  • 2分钟前 │
   └──────────────────────────┘
          ▲ 橙色圆点提示

3. Hover（显示背景 + tooltip）
   ┌──────────────────────────┐
   │  ⚙  │ [↻●] ↑✓  • 2分钟前 │
   └──────────────────────────┘
          ▲ 浅灰背景
     ┌──────────────┐
     │ 文件已更新    │
     │ 点击刷新      │
     └──────────────┘


【同步按钮状态】

1. 未同步
   ┌──────────────────────────┐
   │  ⚙  │  ↻●  ↑   • 2分钟前 │
   └──────────────────────────┘
               灰色图标

2. 已同步（绿色对勾）
   ┌──────────────────────────┐
   │  ⚙  │  ↻●  ↑✓  • 2分钟前 │
   └──────────────────────────┘
               ▲ 绿色对勾

3. 同步中（旋转动画）
   ┌──────────────────────────┐
   │  ⚙  │  ↻●  ↻   • 2分钟前 │
   └──────────────────────────┘
               ▲ 旋转中


【颜色规范】

图标默认色：  #6b7280  (灰色)
Hover 背景：   #f3f4f6  (浅灰)
分割线：      #e5e7eb  (极浅灰)

状态色：
  橙色圆点：  #ff9500  (警告/更新)
  绿色对勾：  #1a7f37  (成功/已同步)
  蓝色高亮：  #3b82f6  (激活/选中)

时间戳：      #9ca3af  (更浅的灰)


【尺寸规范】

按钮尺寸：     32×32px  (可点击区域)
图标尺寸：     16×16px  (实际图标)
圆角：         6px
间距：         8px (按钮之间)
分割线宽度：   1px
字体大小：     12px (时间戳)


================================================================================
推荐实施方案
================================================================================

🏆 首选：方案 A (Figma 风格)

理由：
1. ✓ 最现代、最简洁
2. ✓ 视觉噪音最小
3. ✓ 状态表达清晰（用颜色和图标）
4. ✓ 和主流现代产品风格一致
5. ✓ 扩展性好（未来添加新按钮不会拥挤）

实施步骤：
1. 创建 SVG 图标库
2. 替换 emoji 为 SVG
3. 统一按钮样式（无边框，hover 显示背景）
4. 添加状态指示器（橙色圆点、绿色对勾）
5. 优化间距和对齐
6. 添加 tooltip
7. 添加过渡动画

预计工作量：2-3 小时
