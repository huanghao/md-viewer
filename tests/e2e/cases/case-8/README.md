# Case 8：勾选“同步后立即打开”时成功即关闭

## 目标
验证勾选“同步后在浏览器中打开”时，成功后直接触发打开行为并关闭同步弹窗，不再展示成功状态块。

## 前置条件
- 已打开一个 Markdown 文件。
- 偏好 `openAfterSync=true`。
- 同步执行接口返回成功（测试桩）。

## 步骤
1. 打开同步弹窗，确认“同步后在浏览器中打开”处于勾选状态。
2. 点击同步。

## 预期结果
- 调用 `window.open` 打开文档链接。
- 同步弹窗关闭。
- 不出现弹窗内成功状态展示。

## 失败信号
- 同步成功后弹窗仍停留在页面。
- 未触发打开行为，或仍展示成功状态块。
