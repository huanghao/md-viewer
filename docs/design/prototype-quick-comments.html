md-viewer

一个基于 Web 的 Markdown 查看器，支持实时预览、划词评论、RAG 语义搜索和 PDF 标注。 通过 mdv 命令在本地启动服务，用浏览器访问即可。

核心功能

支持 GitHub Flavored Markdown，包含表格、任务列表和代码块高亮。 内置 KaTeX 数学公式渲染，支持行内公式和独立块公式。 文档目录自动生成，侧边栏展示工作区文件树。

RAG 搜索功能基于本地向量索引，无需联网即可对工作区内所有文档进行语义检索。 划词后可以快速创建评论或 Todo，评论数据持久化存储在本地 SQLite 数据库中。

快速开始

运行 bun install 安装依赖，然后执行 bun dev 启动开发服务器。 构建生产版本使用 bun run build，输出单文件可执行程序 mdv。
