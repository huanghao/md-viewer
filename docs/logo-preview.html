4. 设计特点

✓ 矢量图形

SVG 格式，任何尺寸都清晰锐利，不会模糊

✓ 语义明确

文档图标直观表达 Markdown 查看器的功能

✓ 品牌色彩

蓝色 (#2563eb) 传递专业、可靠、科技感

✓ 现代简洁

参考 Lucide Icons 设计风格，符合当前审美

✓ 视觉统一

页面 logo 和 favicon 保持一致的设计语言

✓ 高识别度

在浏览器标签页中快速识别，提升用户体验
