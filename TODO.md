## 前端代码更新在我浏览器里经常会看不到效果 [✅完成 - 2026-03-01]
包括你改的徽章、logo、文件列表上的状态都经常出不来
复制文件名的按钮也是
- 问题原因：浏览器强缓存了 HTML 页面
- 解决方案：
  1. HTML 添加 no-cache meta 标签
  2. 服务器返回 HTML 时设置 Cache-Control 头
  3. HTML 中添加版本号注释（时间戳）
- 修改文件：src/client/html.ts, src/server.ts
- 完成日期：2026-03-01
