# Logo 设计方案

## 最新设计 (2026-02-28)

### 方案 A: 圆形徽章 ⭐ 当前使用

**设计理念**:
- 完全圆润的设计，无尖角
- 简化的 "M" 字母代表 Markdown
- 参考 Claude、Anthropic 等现代公司的风格

**视觉特点**:
- ✅ 完全圆润，符合现代设计趋势
- ✅ 小尺寸下清晰可辨（16px favicon 效果优秀）
- ✅ 现代、简洁、专业
- ✅ 颜色更柔和：`#3b82f6`（比之前的 `#2563eb` 更温和）

### 设计元素
- **形状**: 完美圆形
- **字母 M**: 代表 Markdown，采用几何化设计
- **颜色**: `#3b82f6` (柔和蓝) + 白色

## 设计演进历史

### 第一版 (2026-02-27)
- **页面 logo**: 📚 emoji
- **问题**: 在不同系统显示不一致，小尺寸不清晰

### 第二版 (2026-02-27)
- **设计**: 方形文档图标 + 折角
- **颜色**: `#2563eb` (较饱和的蓝色)
- **问题**: 用户反馈太方、菱角太锐利，小尺寸下字母不清晰

### 第三版 (2026-02-28) ⭐ 当前版本
- **设计**: 圆形徽章 + 简化字母 M
- **颜色**: `#3b82f6` (柔和蓝)
- **优势**: 圆润、现代、小尺寸清晰

## 技术实现

### 文件结构
```
public/
├── favicon.svg    # 浏览器标签页图标（32x32，蓝色背景）
└── logo.svg       # 页面内嵌 logo（透明背景）
```

### 特性
- SVG 格式，完美缩放
- 内联在 HTML 中，无需额外请求
- 使用 `currentColor` 继承文字颜色
- 响应式设计
