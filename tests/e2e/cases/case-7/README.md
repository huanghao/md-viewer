# Case 7：同步失败后留在同页并支持重试

## 目标
验证单页面同步流程中：首次失败不关闭弹窗，不清空输入；用户可直接重试并在同页看到成功状态。

## 前置条件
- 已打开一个 Markdown 文件。
- 同步执行接口首轮返回失败，第二轮返回成功（测试桩）。

## 步骤
1. 打开同步弹窗并填写标题、父文档。
2. 点击同步（第一次失败）。
3. 不关闭弹窗，直接再次点击同步（第二次成功）。

## 预期结果
- 第一次后显示失败状态块与原始返回。
- 弹窗保持打开，标题和父文档输入值不变。
- 第二次后切换为成功状态块，显示文档标题链接与同步时间。

## 失败信号
- 首次失败后弹窗被关闭或输入被重置。
- 重试后未进入成功状态。
