自建 Text Layer — 三种方案

目标：译文作为新段落紧跟原文，样式一致，不遮挡后续内容。点击选择你倾向的方案。

A
放弃 canvas，纯 HTML 重建
推荐
效果预览
We present a novel approach to neural machine translation that leverages large-scale pretraining.
我们提出了一种利用大规模预训练的新型神经机器翻译方法。
The rapid advancement of deep learning has transformed natural language processing.
深度学习的快速发展改变了自然语言处理领域。
In this paper, we introduce TransFormer-XL, a transformer-based architecture.

优点

  • 真正 reflow，译文插入后后续内容自动下移
  • 文本选中、评论锚点基于段落 index，稳定
  • 可折叠/展开译文

缺点

  • 需要重写渲染层
  • 原文字形与 canvas 背景无法对齐（canvas 已移除）
  • 图片、公式需要 fallback
工作量：
大（3-5天）
B
保留 canvas，译文 div 插入页面流
折中
效果预览
[canvas 背景: 原文渲染位图]
我们提出了一种利用大规模预训练的新型神经机器翻译方法。
[canvas: 第二段原文]
深度学习的快速发展改变了自然语言处理领域。
[canvas: 第三段原文（未翻译）]

优点

  • canvas 保留，原文视觉不变
  • 改动较小，不重写渲染层

缺点

  • canvas 是位图，译文 div 无法真正插入其中
  • 原文和译文在不同层，视觉上割裂
  • 页面滚动时 canvas 和 overlay 需要精确同步
工作量：
中（2-3天）
C
整页翻译后重新生成 HTML 页面
最彻底
效果预览
We present a novel approach to neural machine translation.
我们提出了一种利用大规模预训练的新型神经机器翻译方法。
The rapid advancement of deep learning has transformed NLP.
深度学习的快速发展改变了自然语言处理领域。
In this paper, we introduce TransFormer-XL.
（未翻译）

优点

  • 体验最干净，完全 reflow
  • 可以整页批量翻译，上下文最完整
  • 不依赖 PDF.js 渲染层

缺点

  • 需要先触发整页翻译才能显示
  • 原文字形/排版完全丢失
  • 图表、公式无法还原
工作量：
大（5天+）
已选择方案 A
